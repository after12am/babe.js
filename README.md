babe.js
=======

babe.js is a small programing language works on your browser and now under development.


## A simple example

```html
<script type="text/javascript" src="babe.js"></script>
<script type="text/babe">
class Babe:
    constructor():
        this.name = 'babe.js'
    hello():
        console.log("Hello there, this is " + this.name + "!")

(new Babe()).hello()
</script>
```
